## 二、表结构定义

### 2.1 guestbook（留言表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 主键 |
| name | TEXT | NOT NULL | 留言者昵称 |
| content | TEXT | NOT NULL | 留言内容 |
| email | TEXT | NULL | 可选，用于回复通知 |
| created_at | TEXT (ISO8601) | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| approved | INTEGER (0/1) | NOT NULL, DEFAULT 1 | 是否展示：1=展示，0=待审核 |

**索引：**
- `idx_guestbook_created_at` (created_at DESC) — 按时间倒序展示
- `idx_guestbook_approved` (approved) — 筛选已审核留言

---

### 2.2 contact（联系表单提交记录）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 主键 |
| name | TEXT | NOT NULL | 姓名 |
| email | TEXT | NOT NULL | 邮箱 |
| subject | TEXT | NULL | 主题（可选） |
| message | TEXT | NOT NULL | 留言内容 |
| created_at | TEXT (ISO8601) | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 提交时间 |
| read | INTEGER (0/1) | NOT NULL, DEFAULT 0 | 是否已读：1=已读，0=未读 |

**索引：**
- `idx_contact_created_at` (created_at DESC)
- `idx_contact_read` (read)

---

### 2.3 resume_download_log（简历下载日志，可选）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 主键 |
| ip_hash | TEXT | NULL | IP 哈希（脱敏，用于限流） |
| user_agent | TEXT | NULL | 浏览器 UA（可选） |
| created_at | TEXT (ISO8601) | NOT NULL | 下载时间 |

**索引：**
- `idx_resume_log_created_at` (created_at)
- `idx_resume_log_ip_hash` (ip_hash)

---

## 三、关系说明

- **guestbook** 与 **contact**：无外键关联，各自独立。
- **resume_download_log**：独立日志表，用于统计与限流，无业务关联。

## 五、数据迁移策略

- 使用 Drizzle Kit 生成迁移文件，存放于 `drizzle/migrations/`。
- **首次部署**：执行 `pnpm db:generate` 生成迁移，`pnpm db:migrate` 应用，确保 `migrations/` 与 schema 同步。
- 若采用 `drizzle-kit push`，需在文档中注明并与 schema 保持同步。
- 变更表结构前需与架构师确认，并同步更新本文档。
